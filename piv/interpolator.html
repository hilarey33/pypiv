Cubic interpolator¶

The cubic interpolator is written as its own class.

class piv.interpolator.CubicInterpolator¶

Interpolator Class for the window deformation

This class initializes and performs the interpolation.

Initialization of the interpolator.

__init__(frame, window_size)

The cubic interpolation is performed in compiled c code, therefore the memory for right hand side and left hand side must be allocated and initialized for the matrix. The output is allocated as well.

Parameters:
  • frame – image that is deformed
  • window_size – size of the interrogation window over which is interpolated

It performs the interpolation by calling the interpolate function.

CubicInterpolator.interpolate()¶

Method to perform cubic interpolation of the deformed grid.

interpolate(posx, posy)

Parameters:
  • posx (ndarray) – moved x position of the deformed grid
  • posy (ndarray) – moved y position of the deformed grid
Returns:

ndarray with interpolated sub image shaped to the original shape

The interpolation function itself is not a method of that class, but a part of the interpolator module. The entire module is compiled with Cython in order to decrease the computation time.

piv.interpolator.cubic_interpolation()¶

Function to perform cubic interpolation of a given frame and new coordinates.

cubic_interpolation(np.ndarray[np.double_t, ndim=1] posx,
np.ndarray[np.double_t, ndim=1] posy,
np.ndarray[np.double_t, ndim=1] x,
np.ndarray[np.double_t, ndim=1] y,
np.ndarray[np.double_t, ndim=2] M,
np.ndarray[np.double_t, ndim=1] output,
np.ndarray[np.double_t, ndim=2] frame, int lx, int ly)
Parameters:
  • posx (double_array) – x coordinates the frame is interpolated to
  • posy (double_array) – y coordinates the frame is interpolated to
  • x (double_array) – right hand side memory for interpolation in x direction
  • y (double_array) – right hand side memory for interpolation in y direction
  • M (double_2D_array) – left hand side as a matrix
  • output (double_array) – memory for resulting interpolated sub frame
  • frame (double_2D_array) – initial image which is interpolated
  • lx (int) – width of the frame array
  • ly (int) – length of the frame array
Returns:

interpolated sub frame
